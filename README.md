# Project Title

XML to JSON Converter For Purchase Orders.

## Description

Will convert and XML file to JSON while trimming leading zeros.

## Getting Started

### Dependencies

* node.js
* npm

### Installing

* run npm install

### Executing program
* run below command to get started
```
npm run start
```
* Output JSON File will be located under src/JSONOutput
